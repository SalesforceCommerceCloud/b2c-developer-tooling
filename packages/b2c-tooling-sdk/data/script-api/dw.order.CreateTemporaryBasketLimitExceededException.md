# Class CreateTemporaryBasketLimitExceededException

- [TopLevel.Object](TopLevel.Object.md)
  - [dw.order.CreateTemporaryBasketLimitExceededException](dw.order.CreateTemporaryBasketLimitExceededException.md)

This exception is thrown by [BasketMgr.createTemporaryBasket()](dw.order.BasketMgr.md#createtemporarybasket) to indicate that the open temporary basket
limit for the current session customer is already reached, and therefore no new temporary basket could be created.
