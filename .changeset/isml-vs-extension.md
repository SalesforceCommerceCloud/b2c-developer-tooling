Add ISML language support: file association for `.isml`, TextMate grammar for syntax highlighting (including embedded JavaScript inside `<isscript>` and `${}` expressions), language configuration for comment toggling/bracket pairs/auto-closing, and snippets for common ISML constructs (`isif`, `isloop`, `isinclude`, `isset`, `isscript`, `isdecorate`, `iscontent`, `isprint`, `Resource.msg`, `URLUtils.url`, and more).
