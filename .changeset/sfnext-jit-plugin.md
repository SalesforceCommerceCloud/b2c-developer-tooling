Add `b2c sfnext` topic via JIT plugin (`@salesforce/storefront-next-dev`). Inside a scaffolded Storefront Next project, the CLI automatically uses the project-pinned version; outside projects, it installs the latest on first use. Bootstrap a new project with `npx @salesforce/b2c-cli sfnext create-storefront`.
