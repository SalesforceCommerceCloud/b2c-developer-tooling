Document the new `storefront-next-figma` plugin in the Agent Skills guide, the install snippets (homepage, setup), and the Figma tools setup page. The plugin requires the [Figma MCP server](https://help.figma.com/hc/en-us/articles/32132100833559-Guide-to-the-Figma-MCP-server) and can be installed from the plugin marketplace or with `b2c setup skills storefront-next-figma`.
